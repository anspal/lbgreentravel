                    About this Web Map
                

                
Created by: Anshu Pallav

                
This website is developed as a part of final project for the MS GIS Batch 2017 from California State University Long Beach. It is has been created using open source enterprise GIS architecture. This website shows the sustainable travel options available. The code to create a buffer and use it make selection using Mapbox JS api and Turf was downloaded from GitHub and modified for this project."

                

                    Instructions
                    
 Click a location or enter an address to find bike share location in 0.5 miles. The blue marker can be dragged across the map and buffer area can be changed by scrolling.   
                

                

                    Built with...
                    
 Mapbox JS, Turf, jQuery, GeoServer, PostgreSQL, PostGIS, AWS
